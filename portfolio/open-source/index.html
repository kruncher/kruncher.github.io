Open Source

I’ve released a number of packages as open source over the past few years; primarily relating to development with Unity. Some of the packages have been integrated into popular assets on that Unity Asset Store which are used in the development of quite a number of games.

Projects on Bitbucket

Originally I hosted open source projects on Bitbucket which are standalone and can be imported into any project by copying the files or using one of the pre-made Unity packages that I provided:

https://bitbucket.org/rotorz/

Projects on GitHub

More recently I switched to using npm to split packages and manage their dependencies so that it’s easier to maintain and reuse them. To avoid confusion I decided to host these packages on GitHub instead:

https://github.com/rotorz
