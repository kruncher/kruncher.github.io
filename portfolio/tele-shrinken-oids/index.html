Tele-shrinken-oids

A mobile arcade game where the player must react quickly to counter spawning objects with paper-rock-scissors like interactions.

I designed and created this game with Unity and created the artwork using Blender and Photoshop with a custom shader which adds a rimlit effect to the objects. There is a lot of smooth animation in this game with various objects moving between screens during transitions, etc. The game features achievements, leaderboards and allows the player to pay whatever they want for the game rather than having intrusive adverts. Players are rewarded hats for completing various accolades which can be selected using the hat selector interface.
